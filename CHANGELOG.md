# Changelog

All notable changes to EchoStory will be documented in this file.

## [1.0.0] - 2026-03-10

### Added
- Initial release for 2026 Hackathon
- Real-time voice interaction with Gemini Live API
- Live Vision integration for object detection
- Interleaved multimodal output (text + audio + images)
- Imagen 3 integration for child-friendly illustrations
- WebSocket-based bidirectional communication
- FastAPI backend with async support
- Next.js frontend with TypeScript
- Sensory-friendly UI with emotion-based backgrounds
- Framer Motion animations
- Story segment persistence in Firestore
- Cloud Storage integration for assets
- Docker containerization
- Google Cloud Run deployment scripts
- Automated deployment with Cloud Build
- Comprehensive documentation

### Core Features
- Low-demand interaction design
- Interruption handling
- Session-based story tracking
- Proactive vision-based story integration
- Emotion-aware UI adaptation
- Real-time audio streaming
- Camera frame analysis
- Parent-accessible story journal
